# Python CPF CNPJ generator and validator

## Installation
- `git clone git@github.com:wasosky313/python_cpj_cnpj_generator.git`
- `cd python_cpj_cnpj_generator.git`
### Install python dependencies
- `poetry shell`
- `poetry install`

## How to run
`python main.py"`
